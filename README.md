# One-Shot-Learning-Based-Face-Recognition-System
**摘要**:
  本研究旨在深入探討基於單一樣本學習技術的人臉辨識點名系統。鑒於資料集的數量不足是目前深度學習領域的一大挑戰,傳統的機器學習策略在少樣本數據下難以達到理想效果。在參考過去學者提出的相關文獻後,我們採用了數據增強、特徵學習、遷移學習,再結合三重網路以及simCLR,透過不同以往的做法,我們引入集成學習策略,目的在於結合不同模型,進一步增強模型的預測能力和穩定性。實驗結果顯示,限定正臉照且人數為 20 的條件下,相較於單一模型之預測,集成方法確實為整體精確度帶來 5% 至 20% 提升,且在預測自行蒐集的資料集以及 CelebA 資料集,模型精確度達到了約 80%至 90%。此實驗數據支持本研究所提出之方法確實能夠增強系統的泛化性能,並提高了模型的預測能力及穩定性。

**關鍵字**:  單一樣本學習、遷移學習、孿生網路、三重網絡、simCLR、集成學習。
